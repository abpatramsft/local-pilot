# BMI Calculator Web Application

This is a simple web application to calculate Body Mass Index (BMI) using height and weight inputs.

## Features
- Input height (cm) and weight (kg)
- Calculates BMI instantly
- Displays BMI value and category (Underweight, Normal, Overweight, Obese)
- Responsive and user-friendly interface

## Usage
1. Open `bmi-calculator.html` in your web browser.
2. Enter your height in centimeters and weight in kilograms.
3. Click the "Calculate BMI" button.
4. View your BMI and its category below the form.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## No Installation Required
Simply open the HTML file in any modern web browser. No dependencies or server setup needed.
